# Visuals

This folder will contain charts, diagrams, and screenshots related to the project.

## Suggested Visuals
- **Password Strength Chart:** A bar chart showing the frequency of weak vs. strong passwords in the mock dataset.
- **Trust Perception Diagram:** A diagram illustrating how users interpret browser security indicators (padlock, HTTPS).
- **Phishing vs. Secure Page Screenshots:** Side-by-side screenshots of the secure.html and phish.html demo pages.
- **User Behavior Flowchart:** A flowchart showing typical user decision-making when encountering security indicators.

## Purpose
Visuals will help communicate findings more clearly and provide evidence for the analysis in the final report.
